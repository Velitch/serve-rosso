# STUDIO


SeRvə RosSo è un'opera versatile. In piazza è [interattiva](https://github.com/Velitch/serve-rosso/blob/main/studio/ableton_live_m4l.md), al museo [si adatta](https://github.com/Velitch/serve-rosso/blob/main/studio/max_msp.md). Comunica con la [segreteria](https://github.com/Velitch/serve-rosso/blob/main/studio/segreteria.md) per essere sempre aggiornatə sul meteo, chiamate e messaggi, anche se ci tiene da sè a tenere in ordine la [libreria](https://github.com/Velitch/serve-rosso/blob/main/studio/bibliografia.md), [videoteca](https://github.com/Velitch/serve-rosso/blob/main/laboratorio/valigia/scatola/proiettore.md) e l'[agenda](https://github.com/Velitch/serve-rosso/blob/main/studio/to_do.md).
